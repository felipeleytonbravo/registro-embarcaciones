# Registro de Embarcaciones 

Programa simple en Python que permite registrar embarcaciones y mostrarlas en pantalla.

##  Funcionalidades

- Registrar embarcación
- Guardar datos: nombre, tipo, capacidad, capitán
- Mostrar todas las embarcaciones registradas

##  Cómo ejecutar

python registro_embarcaciones.py


##  Ejemplo de uso

1) Registrar embarcación  
2) Mostrar registro  
3) Salir  

##  Tecnologías

- Python 3
